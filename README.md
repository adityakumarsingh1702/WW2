A visually immersive **parallax scrolling website** that narrates key events from World War II, including the Battle of Stalingrad, Operation Barbarossa, and the Soviet victory. Built using **pure HTML and CSS**, this project focuses on storytelling through design using full-screen parallax images and clean section transitions.

 Features

- Parallax scrolling effect using CSS `background-attachment: fixed`
- Multiple full-screen background images representing historical events
- Smooth transitions between dark and light content sections
- Responsive text positioning centered over parallax images
- Minimal, clean, and educational content layout
